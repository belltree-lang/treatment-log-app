# Issue #1208 共有用サマリー

## 変更点
- `billingJson` の各行から非永続の派生ビュー（insurance/self-pay/mixed）を作成し、集計で保険合計・自費合計を算出できるようにしました。
- 請求一覧の合計カードに「保険合計 / 自費合計」をサブ表示として追加しました（既存の合計表示は維持）。
- サブ表示用のスタイルを追加しました。

## 理由
- `billingJson` の保存形式や Sheets/PDF ロジックを変更せず、表示・集計のみで保険請求と自費請求を分離する要件に対応するためです。

## 影響範囲
- 請求一覧 UI（合計カードのサブ表示追加）
- フロントエンドの集計処理（派生ビューによる保険/自費合計の算出）
